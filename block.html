1. Difference between map find filter forEach ?

Answer : Map, find, filter, forEach these are array methods, These methods are described below.....

  • 1. Map method : The map() method return an array . It is used to create a new array, access each element of the array separately using a loop thro throw
  • 2. Filter method : Filter is an array method, given a condition and based on that condition,only the elements that match will be returned as an array
  • 3. Find method : The find mathod is the younger brother of the filter method .Using the find method only one element is returned, based on the condition it will find the first match and return it without going any further
  • 4. ForEach method : If we don't need to return anything from the array then we will use the forEach method

3. What is Event loop ?

Answer :

  • Event loop is a programming construct that continuously looks for some type of event or activity and then reacts to it. It is used to process events in a timely fashion, such as user input, system events, and messages from other programs or threads. It is core to the design of many software systems, including operating systems, windowing systems and even web browsers.

QuizTimer Function & ChooseQuiz Function

1. How does the quizTimer function work ?


  • 1. The quizTimer function takes one parameter, dismiss, which is a boolean.
  • 2. If the dismiss parameter is true, the function will clear the timer interval, otherwise it will set an interval to count the seconds.
  • 3. Inside the interval, a countHtml variable is set to select the #count element in the HTML document.
  • 4. The count is then converted to minutes and seconds and stored in the min and countSec variables.
  • 5. The innerHTML of countHtml is then set to the min and countSec variables.
  • 6. The count is incremented by one.
  • 7. If the count is greater than 60, the class of countHtml is changed to either red-300 or red- 700.
  • 8. Finally, the interval is cleared or set again, depending on the value of dismiss.

1. How does the chooseQuiz function work ?

Answer :

  • 1. The chooseQuiz function takes two parameters, an index and a givenAns.
  • 2. It then checks if an answer corresponding to the given index exists in the answers array.
  • 3. If an answer does exist, the function will loop through the answers array to find the answer that corresponds to the given index.
  • 4. If the answer is found, it will then be spliced from the answers array, and the new answer with the givenAns is added in its place.
  • 5. If an answer does not exist, the new answer with the givenAns is added to the answers array.
  • 6. Finally, the displayAnswers function is called and the answers array is passed in as a parameter.
